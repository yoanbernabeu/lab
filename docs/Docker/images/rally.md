# Rally

## Description

Alternative au sondage doodle auto-hébergé. Pratique pour organiser des réunions ou des événements.

## Installation

```bash
git clone https://github.com/lukevella/rallly.git
cd rallly
cp sample.env .env
docker-compose up -d
```
